# خلفيات الموقع الجاهزة (Background presets)

هنا تضع فيديوهات (أو صور) الخلفية الجاهزة التي يختار منها المكاتب.
تُخدَّم من CDN التطبيق ولا تستهلك أي مساحة في قاعدة البيانات/التخزين.

## كيف تضيف خلفية جديدة (خطوتان)

1) أسقط ملف الفيديو هنا داخل هذا المجلد:
   platform/public/backgrounds/my-video.mp4

2) سجّله في الملف:
   platform/src/lib/bg-presets.ts
   بإضافة سطر داخل المصفوفة BG_PRESETS:

   { id: "my-video", name: "اسم يظهر للمكتب", src: "/backgrounds/my-video.mp4" },

   - id: معرّف فريد (إنجليزي بدون مسافات).
   - name: الاسم العربي الذي يظهر في المحرّر.
   - src: المسار يبدأ بـ /backgrounds/ ثم اسم الملف بالضبط.

بعدها تظهر الخلفية تلقائياً في المحرّر ضمن «خلفية الموقع → فيديو طبيعي → اختر خلفية جاهزة».

## لجعل الخلفية تعمل بحركة التمرير (مثل أوتاد)
تحتاج إطارات مُعدّة مسبقاً بمعدل 16 إطار/ثانية في مجلد باسم الـ id:

  ffmpeg -i my-video.mp4 -vf "fps=16,scale='min(1280,iw)':-2" -q:v 4 -start_number 0 my-video/f%03d.jpg

ثم في bg-presets.ts اضبط frameCount بعدد الإطارات الناتج. تظهر الخلفية حينها ضمن
«خلفية الموقع → حركة مع التمرير → اختر خلفية جاهزة».

## نصائح
- يُفضَّل mp4 (أو webm) قصير 5–12 ثانية، أفقي، وأقل من ~5MB لسرعة التحميل.
- تجنّب الأسماء العربية أو المسافات في اسم الملف؛ استخدم أحرفاً إنجليزية وشرطات: tower-night.mp4
- يمكن أيضاً وضع صورة ثابتة (jpg/png) كخلفية بنفس الطريقة.
