# خلفيات صفحة الهبوط (Landing backgrounds)

خلفيات خاصة بالصفحة الرئيسية للمنصّة فقط (منفصلة عن خلفيات المكاتب).
يختار منها السوبر أدمن من: لوحة الإدارة → «صفحة الهبوط» → خلفية الصفحة.

## كيف تضيف خلفية (خطوتان)

1) أسقط ملف الفيديو هنا:
   platform/public/landing-backgrounds/hero1.mp4

2) سجّله في الملف:
   platform/src/lib/landing-presets.ts
   بإضافة سطر داخل المصفوفة RAW:

   { id: "hero1", name: "خلفية الهبوط 1", src: "/landing-backgrounds/hero1.mp4", frameCount: 0 },

   - id: معرّف فريد إنجليزي بدون مسافات.
   - name: الاسم العربي الذي يظهر في المحرّر.
   - src: المسار يبدأ بـ /landing-backgrounds/ ثم اسم الملف بالضبط.
   - frameCount: يبقى 0 إن أردت «فيديو متحرك» فقط. للحركة مع التمرير، استخرج الإطارات (أدناه) وضع عددها.

## لتفعيل «الحركة مع التمرير» (إطارات 16fps)
استخرج الإطارات إلى مجلد باسم الـ id:

  ffmpeg -i hero1.mp4 -vf "fps=16,scale='min(1280,iw)':-2" -q:v 4 -start_number 0 hero1/f%03d.jpg

ثم اضبط frameCount في landing-presets.ts بعدد الإطارات الناتج.

## نصائح
- mp4 (أو webm) قصير 6–15 ثانية، أفقي، وأقل من ~5MB لسرعة التحميل.
- أسماء إنجليزية بدون مسافات: hero-night.mp4
- يمكن أيضاً وضع صورة ثابتة (jpg/png) كخلفية بنفس الطريقة.
